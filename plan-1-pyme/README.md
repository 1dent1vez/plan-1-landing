# landing-mobile-first

Landing mobile-first con HTML semantico, CSS por tokens y JS progresivo. El sitio funciona sin JS y mejora la experiencia con menu accesible y acordeones.

## Como abrir

- Abre `index.html` en el navegador.
- Edita `config/config.json` para cambiar textos y enlaces.

## Validacion rapida

- Mobile: 375, 390 y 414 px.
- Desktop: >= 1024 px.
- Lighthouse objetivo: Performance >= 90, Accessibility >= 90, Best Practices >= 90, SEO >= 90.

## Estructura

- `index.html`: estructura semantica, nav accesible y secciones.
- `src/styles/*.css`: tokens, base, componentes y secciones.
- `src/js/*.js`: bindings, menu accesible y motion progresivo.
- `config/config.json`: contenido editable.
